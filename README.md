# Advent of Code 2025

Functional Scala solutions for [Advent of Code 2025](https://adventofcode.com/2025).

## Stack

- Scala 3.3 with [scala-cli](https://scala-cli.virtuslab.org/)
- [Cats](https://typelevel.org/cats/) for functional abstractions

## Running

```bash
scala-cli run dayXX.scala < inputs/dayXX.txt
```
